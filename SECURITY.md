# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GEMRISER, please email security@gemriser.dev.

Do not disclose the issue publicly until we've had a chance to address it.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Active |
| < 1.0   | ❌ (legacy v0.5.3 is deprecated) |

## Disclosure Timeline

We aim to respond within 72 hours and release a fix within 90 days of disclosure.
